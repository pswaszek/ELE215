ELE 215 - Submitting your Work

Documentation of your work in ELE 215 is to be submitted via Brightspace. Specifically,
  • Labs 1, 2, 4, and 5 have just a single- or double-sided summary page that should be uploaded as a single pdf document.
  • Labs 6, 7, 8, 9, 10, and 11 have a summary page(s) and additionally require a graph or graphs. These graphs should be on just one separate page and be included as part of the single pdf document.
  • Lab 3 requires a MatLab program which should be uploaded as a single script file.
  • Lab 12, the Black Box, requires a brief report; format will be discussed later in the semester.
When uploading this single document/script to the ELE 215 Brighspace site, please use the following naming convention for your file:
  • In general, the name should be the concatenation of which effort it represents (i.e. lab number) followed by student name(s), all separated by underscores.
  • For student names, please shorten then to 6 letters - 5 for the last name (padded by x's if needed) plus the first initial, using capitals for the first letter of the last name and the initial, no spaces.
  • For example, let's consider student Thomas MacGreggor and his effort on Lab 2 (in which you each work individually). He would scan his summary page to

            Lab_2_MacgrT.pdf

    and submit this file via Brightspace. In a similar way, student Barbara Jobb would submit the file

            Lab_2_JobbxB.pdf

    (the extra x after Jobb extends her last name to 5 letters). For Lab 3 their individual MatLab scripts would be named

            Lab_3_JobbxB.m

    and

            Lab_3_MacgrT.m

    For joint work, say Lab 1, the single submission would be called

            Lab_1_JobbxB_MacgrT.pdf

    putting the names in alphabetical order by last name for joint work.
  • Which student submits the joint work? Go with the one whose last name appears first alphabetically; hence, for the above example Barbara Jobb submits.
A few of these activities are individual, but most are for pairs of students:
  • Lab 2, 3, and 6 are individual work so should be submitted individually.
  • Lab 1, 4, 5, 7, 8, 9, 10, and 11 are intended for pairs of students (and I will preassign pairs, rotating you with other class mates); in these cases I expect to receive only one submission from each pair. If for some reason your assigned lab partner is unavailable, you should work alone; working in triplets or modifying my pairings will be penalized.
  • For the final lab (12) you may work with a partner of your choice.
How can you generate a pdf file?
  • The printers at URI's library scan
  • Smartphone apps such as Notes for iPhones do a reasonable job; just be sure to submit something that is high quality.
